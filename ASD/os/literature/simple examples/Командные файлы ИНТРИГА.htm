Командные файлы



Командные файлы - это программы, которые позволяют пользователям MS-DOS и Windows создавать файл, чтобы выполнить длинные задачи быстро. Такие как, открытие определенных часто используемых программ, удаление или управление файлами, и т.д. Простые командные файлы не требуют никаких специальных программных навыков и обычно могут быть созданы, зная ниже перечисленные команды и команды DOS.

Простой пример командного файла для тех кто знает немного Windows, связать командный файл с ярлыком в Windows. Командные файлы как ярлыки(shortcuts) - маленький файл, который используется, чтобы запускать, часто используемые программы.

Другой пример очень хорошо известного командного файла - autoexec.bat, который является просто файлом загрузки, загружается каждый раз, когда загружается компьютер. В теле этого файла содержится различные команды, чтобы загрузить различные программы и драйверы в память каждый раз при загрузке компьютера.

Создание командного файла

Для создания командного файл сначала запустите консоль MS-DOS и следуйте следующим инструкциям:
В MS-DOS наберите edit test.bat и нажмите ENTER. Если если все было сделано верно, то на "синем" экране наберите следующее:
pause
dir c:\windows
dir c:\windows\system

Как только вышеупомянутые три строки набраны в "синем" экране, выберите меню Файл и Выход, на предложение сохранения ответить "да".

После возврата в MS-DOS наберите test. Когда Вы нажмете ENTER, начнется исполнение test.bat. Поскольку первая строка - пауза, Вам будете сначала предложено нажать клавишу. Как только Вы нажмете клавишу, командный файл будет выполнять строку за строкой. Если бы Вы желаете добавить больше строк к этому командному файлу, Вы просто набираете, "edit test.bat", чтобы редактировать файл снова.

Исполняемые команды:

@ - Запрещает вывод команды следующей за символом.

@ECHO OFF - Запрещает вывод команд в последующих строках.

%1 - Процент и цифра, начиная с единицы, позволяет добавлять переменные в команды.
Например myname.bat:
echo Hello %1

За пускаем myname Vasia

Вывод:

Hello Vasia

Замечание: Можно продолжить %2, %3, и т.д.

:LABEL - создает метку, переход к которой можно осуществить командой GOTO.

CALL - используется для вызова другого командного файла. После завершения исполнения вызванного файла будет продолжено выполнение основного файла. Если вызывается несуществующий файл будет выдана ошибка.

CLS - Очищает экран DOS.

ECHO - выводит сообщение на экран. "ECHO Hello World" выведет на экран Hello World при выполнении.
Если вам нужна пустая строка набирайте "ECHO."

EXIT - закрытие окна MS-DOS.

GOTO LABEL - используется для перехода к конкретной метке.

IF - используется для сравнения значений.
Например:
IF ERRORLEVEL ==3 GOTO MYLABEL

PAUSE - пауза. Предлагает пользователю нажать клавишу

REM - позволяет добавить комментарии в командный файл.

SHIFT - изменяет последовательность параметров командного файла. Дополнительную информацию можно узнать набрав SHIFT /? в DOS
